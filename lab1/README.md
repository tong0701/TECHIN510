# TECHIN 510 — Lab 1: Agentic Coding

Two Streamlit apps built for the Week 1 lab.

## Purchase Tracker

Helps GIX students submit purchase requests and helps the program coordinator review and export them.

```bash
cd purchase-tracker
pip install -r requirements.txt
streamlit run app.py
```

## GIX Wayfinder

Helps new GIX students find campus resources like the makerspace, printing, bike storage, etc.

```bash
cd wayfinder
pip install -r requirements.txt
streamlit run wayfinder.py
```
